Version 0.6-RC1

January 16, 2019

Update 0.6 is a complete rewrite of Notch'd. Instead of hacking everything together, most of the tweaking is now done by hooking MobileGestalt (MGCopyAnswer in particular).

This update should now work with almost any app as this is the most "native" way possible to enable iPhone X features. No more weird frame manipulations on a somewhat per app basis.

However, there is something very important with RC1:
This release unfortunately still includes a bug that puts your device at a huge security risk. For some reason, your device may stay unlocked even though it should be locked with a passcode or Touch ID. If you experience this bug, disable the Face ID latch as soon as possible! Don't tell me I didn't warn you! IT'S RIGHT HERE! I really hope I can fix this bug in a later release.

Anyways, here's the changelog you've been waiting for:

  • Completely rewritten, should now work with any app
  • Various speed improvements thanks to single MobileGestalt hook
  • Even more iPhone X detail (Face ID icon while unlocking)
  • Fixed widgets being pushed down when on home screen
  • Face ID latch now scrolls with notifications

Also, here's a list of currently known issues (I know it's longer than the changelog, but it's still RC1):

  • Severe: Face ID latch may keep the device unlocked at all time
     Workaround: Restart SpringBoard or disable "Face ID Latch"
  • Critical: Per-App settings not implemented
  • Major: Currently incompatible with Discord (maybe other apps)
  • Minor: Layout incompatibility with StackXI
  • Minor: Face ID latch is not hidden if unlocking with Touch ID fails within 5 attempts
     Workaround: Turn off screen and try again
